# Personal Portfolio Website

This project is a simple personal portfolio website developed using HTML, CSS, and JavaScript.  
The website introduces me, my interests, and some of the projects I have worked on.

## Author
Parilaksh Thakur  
B.Tech Computer Science Engineering Student  
Rayat Bahra University

## Features
- Responsive portfolio layout
- Navigation bar for easy section access
- About section describing my background
- Projects section showcasing my work
- Contact form for user interaction
- Interactive feature to add new benefits of bike riding
- Live date and time displayed in the footer

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run the Project

1. Download or extract the project files.
2. Open the project folder.
3. Locate the file named **index.html**.
4. Double-click the file or open it in any web browser (Chrome, Edge, Firefox, etc.).
5. The portfolio website will load and can be viewed locally.

## Description

The homepage of the website highlights my hobby of bike riding and allows users to add additional benefits dynamically using JavaScript.

The about section briefly describes my academic background and technical interests.

The projects section showcases some example projects including a weather app, snake game, and a bingo game.

The contact section provides a simple form for visitors to send messages.
